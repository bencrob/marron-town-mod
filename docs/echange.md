# 💱 Échange — ressources contre points

[⬅️ Retour au guide](README.md)

Une fois le **plafond d'XP atteint** (90 points), l'échange devient ta **source principale de
points** pour remplir tes autres compétences. Ouvre le Grimoire 📕 → **Échange**.

Tu **donnes des ressources**, tu **reçois des points de compétence**. 🪙

## 🔁 Offres fixes (toujours disponibles)

| Tu donnes | Tu reçois |
|---|---|
| 💎 64 Diamants | +5 points |
| 💚 64 Émeraudes | +5 points |
| 🟫 32 Débris antiques | +3 points |

Répétables autant que tu as les ressources.

## ⭐ Bonus du jour (1× par jour)

Chaque jour, une **offre bonus** différente apparaît (ex. **128 Pains → +10 points**,
**64 Bûches → +1 point**…). Elle est **utilisable une seule fois par jour et par joueur**, puis
se renouvelle automatiquement. Le menu affiche le temps avant le prochain renouvellement. ⏳

> 💡 Garde tes diamants/émeraudes pour l'échange plutôt que pour la boutique — ce sont les
> meilleures sources de points sur la durée. Le **bonus quotidien** est presque toujours rentable. 🌰
