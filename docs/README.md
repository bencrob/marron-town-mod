# 🌰 Marron Town — Guide du joueur

Bienvenue ! **Marron Town** transforme ta partie de Minecraft en aventure **RPG** :
ton XP te rapporte des **points de compétence**, tu les dépenses dans **4 arbres** pour
devenir plus rapide, plus fort, plus résistant ou meilleur mineur, et tu claques tes points
dans une **boutique** qui change tout le temps. 🎮

---

## 🚀 Bien démarrer

1. **Fabrique le Grimoire des Compétences** 📕 sur une table de craft —
   **bâton + papier + or brut**, alignés horizontalement :

   ```
   🪵 📄 🟡     →  📕 Grimoire
   ```
   (🪵 Bâton · 📄 Papier · 🟡 Or brut, côte à côte sur une ligne)

2. **Tiens le Grimoire en main et fais un clic-droit** (ou appui long sur mobile/manette).
   → Le menu s'ouvre. 🪄
3. Tu démarres avec **10 points** ; ensuite, tous les **5 niveaux** d'XP → **1 point** (jusqu'à un plafond, voir [Progression](progression.md)). ✨
4. Ouvre le Grimoire → **Compétences** → choisis un arbre → dépense tes points !

> 💡 Le Grimoire est **conservé quand tu meurs** : si tu le perds, tu le retrouves au réveil.

## 🧭 Le menu en un coup d'œil

- **⚔️ Compétences** — choisis un arbre et monte tes niveaux (1 point = 1 niveau).
- **🏪 Boutique** — achète des objets avec tes points (rotation toutes les 12 h).
- **📊 Ma Fiche** — ton résumé : niveaux, points gagnés / dépensés / disponibles.
