# 🎨 Customisation & thème

[⬅️ Retour au guide](README.md)

## 📕 Choisir la couverture de ton grimoire

Ouvre le Grimoire 📕 → **Customisation**. Tiens ton grimoire en main, puis choisis une
**couverture** parmi plusieurs : Marron (défaut), Violet, Émeraude, Or, Redstone, Pierre Moussue.

La couverture choisie **change l'apparence de ton grimoire** partout :
- 🖐️ en main ;
- 🎒 dans l'inventaire et la barre de raccourcis ;
- 🌍 quand il est jeté au sol.

> Chaque couverture garde le **nom du propriétaire** (« Grimoire de … »).

## 🌿 Thème « Mossy Stone »

Dans **Paramètres**, tu peux activer le thème **Mossy Stone** :
- le titre du menu passe au vert mousse 🌿 ;
- ton grimoire prend la couverture **Pierre Moussue**.

Réversible à tout moment (re-clique pour désactiver).

> ℹ️ Sur Bedrock, l'apparence interne des menus (boutons, fond) est imposée par le jeu : le thème
> agit sur les éléments qu'un add-on peut changer (titres, couverture, couleurs de texte).
